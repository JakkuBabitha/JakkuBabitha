- 🧠 Exploring GenAI applications — LangGraph, LLMs, and the Groq API
- 📊 Comfortable across the full ML pipeline: EDA → feature engineering → modeling → deployment
- 🔍 Interested in anomaly detection, time-series models, and explainable AI (XAI)
- 🎯 Actively preparing for Data Science / ML placements in Bengaluru

## 🚀 Featured Projects

### 🧾 [AI-Powered Resume Builder](https://github.com/JakkuBabitha)
End-to-end GenAI app using LangGraph + Groq (`llama-3.1-8b-instant`) + Streamlit, built across 9 development phases with real production debugging — JSON parsing failures, token rate limits, DOCX table-reading bugs, session state issues.
`LangGraph` `Groq` `Streamlit`

### ⚙️ [Predictive Maintenance — Vibration Data](https://github.com/JakkuBabitha)
Reproduced and extended an unsupervised ML pipeline (PCA T²/SPE, K-Means, Hierarchical Clustering, Fuzzy C-Means, GMM) with newer anomaly detection models — Isolation Forest, One-Class SVM, MLP Autoencoder, and an LSTM Autoencoder — for exhaust-fan vibration anomaly detection.
`TensorFlow` `Keras` `Scikit-learn`

### 📡 Real-Time Predictive Maintenance Framework
Kafka + Spark Structured Streaming + LSTM on the NASA CMAPSS dataset, built as a real-time Big Data Analytics project package.
`Kafka` `PySpark` `LSTM`

### 🔋 Smart Energy Consumption Monitoring
Stream processing micro-project using Kafka and Spark Streaming for real-time energy analytics.
`Kafka` `Spark Streaming`

### 📚 Explainable AI (XAI) — Systematic Literature Review
