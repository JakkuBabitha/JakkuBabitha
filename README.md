### ⚙️ Predictive Maintenance — Vibration Anomaly Detection
**Problem:** Detecting early-stage mechanical failure from exhaust-fan vibration sensor data.
**Approach:** Reproduced a published unsupervised pipeline (Amruthnath & Gupta, 2018) — PCA T²/SPE, K-Means, Hierarchical Clustering, Fuzzy C-Means, GMM — then extended it with newer models: Isolation Forest, One-Class SVM, MLP Autoencoder, and an **LSTM Autoencoder** (TensorFlow/Keras), identified as the strongest performer.
**Deliverables:** Full notebooks, a 19-slide technical presentation, and a 10-paper literature review with gap analysis.
**Stack:** `TensorFlow` `Keras` `Scikit-learn` `Time-Series Analysis`
