## 🚀 Featured Projects

### 🧾 AI-Powered Resume Builder
Agentic GenAI app: parses a resume, matches it to a job description, rewrites content for fit. LangGraph + Groq (`llama-3.1-8b-instant`) + Streamlit, shipped across 9 build phases with real debugging (JSON parsing failures, token rate limits, DOCX bugs, session state).

### ⚙️ Predictive Maintenance — Vibration Anomaly Detection
Reproduced Amruthnath & Gupta (2018)'s unsupervised pipeline (PCA T²/SPE, K-Means, GMM, Fuzzy C-Means), then extended it with Isolation Forest, One-Class SVM, and an LSTM Autoencoder — the strongest performer of the set.

### 📡 Real-Time Predictive Maintenance Framework
Kafka + Spark Structured Streaming + LSTM on NASA's CMAPSS turbofan degradation dataset — moved the above problem from batch to streaming.
